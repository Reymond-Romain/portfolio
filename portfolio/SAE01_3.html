Installation d'un poste pour le développement

Enoncé du projet

Dans cette SAE, nous devions installer une machine virtuelle Debian 11 avec l'interface KDE/Plasma ainsi qu'installer un JDK et le logiciel de code Intellij IDEA

Ce que j'ai fait

Tout d'abord, j'ai installé le système d'exploitation Debian 11 en suivant les consignes qui nous étaient données. Ensuite, je me suis occupé d'installer le JDK, toujours en suivant à la lettre les consignes données. Enfin, j'ai installé l'application IDEA Intellij via le terminal de la machine virtuelle.

Les compétences utilisées

Dans cette SAE j'ai utilisé des compétences de langage UNIX pour installer Debian ainsi que le JDK. J'ai également utilisé des compétences de recherche pour trouver comment continuer les Installations
