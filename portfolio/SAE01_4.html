Création d'une base de données

Etude du naufrage du Titanic

Enoncé du projet

Lors de cette SAE, nous devions créer en binôme une base de données à partir d'un fichier tableur sur les passagers du Titanic, puis l'utiliser pour déterminer quelles personnes avaient le plus de chances de survie au moment du naufrage

Ce que nous avons fait

Tout d'abord, nous avons utilisé un fichier Excel pour créer un schéma SEA (représente les connexions entre chaque tables de la base de données) ainsi qu'une justification des choix qu'on a fait à ce sujet. Ensuite, nous avons justifié le schéma relationnel qui nnous a été fourni et avons implémenté cette base de données en ligne sous PostgreSQL. Nous l'avons ensuite testé Enfin, nous avons testé des requêtes afin de récolter des informations sur le naufrage afin de pouvoir dire quel type de personne avait le plus de chances de survie lors du naufrage du Titanic.

Les compétencs utilisées

Dans cette SAE, nous avons mis en oeuvre des compétences d'analyse pour trouver les informations dont nous avions besoin à l'intérieur du fichier Excel. Nous avon également utilisé des compétences en SQL pour réaliser toutes une base de données pour traiter ces informations.
